NORMALISASI DATABASE
apa itu
Normalisasi database adlah proses pengorganisasian dan penataan database untuk mengurangi redundansi data,
anda dapat menghindari kesalahan inpout data, penghapusan tidak disengaja, dan anda juga memudahkan proses pembaruan data

manfaat
mencegah masalah dan membuat pemeliharaan database lebih mudah ini akan menghindari pemborosan sumber daya dan uang
menyebabkan berkurangny ruang penyimpanan
normal form itu form yang beraturan

Class diagram digunakan untuk menampilkan struktur model data. Class diagram terdiri atas :
